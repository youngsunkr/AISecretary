# 📋 작업 브리프

**원 명령:** 최신 .net으로 평가시스템 구축해주고 ml.net도 적용해줘

## 요약
최신 .NET 환경을 활용하여 '평가 시스템' 구축 및 ML.NET 통합에 대한 요청입니다. 개발 착수 전, 기술적 아키텍처 설계와 비즈니스 목표 정의가 선행되어야 합니다.

## 분배
- **🔍 Researcher**: .NET 9 이상의 버전에서 ML.NET을 효과적으로 결합할 수 있는 평가 시스템의 최신 모범 사례(Best Practice) 아키텍처 패턴(예: Clean Architecture/CQRS 기반의 서비스 레이어 설계, 비동기 처리 방식 등)를 조사하고, 기술적 요구사항과 제약 사항을 정리해줘.
- **💼 현빈**: 구축될 '평가 시스템'이 어떤 목적으로 사용되어야 할지 가정(예: 코드 품질 평가, 성능 병목 지점 분석 등), 이 시스템을 통해 측정하고 비즈니스를 개선할 수 있는 핵심 KPI 3가지와 구체적인 수익화 가능 영역을 정의해줘.
- **💻 코다리**: Researcher가 제공한 최신 아키텍처 패턴과 Business가 정의한 KPI를 바탕으로, .NET Core 기반의 모듈식 평가시스템 초기 기술 스택(Technology Stack) 및 핵심 데이터베이스 구조(Schema Draft)를 설계하고, 가장 먼저 구현해야 할 API 엔드포인트 3개를 정의해줘.
