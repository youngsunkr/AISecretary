# 📋 작업 브리프

**원 명령:** 회사 홈페이지 만들어줘

## 요약
사용자가 회사 홈페이지 제작을 요청했습니다. 이는 회사 정체성(에프원소프트, .net 컨설턴트)을 반영한 웹사이트 구축 작업입니다.

## 분배
- **💻 Developer**: 회사 웹사이트 개발을 위한 기술 스택 및 아키텍처 초안을 작성해 주세요. (필수 포함 요소: Clean Architecture, CQRS/Mediator Pattern, .NET 9+ 기반, PostgreSQL 연동, Web API 설계 포함)
- **🎨 Designer**: 에프원소프트의 '회사' 웹사이트를 위한 브랜드 비주얼과 디자인 브리프를 작성해 주세요. (타깃 청중: 고객, 회사 / 톤앤매너: 전문적이고 신뢰감을 주는 컨설팅 회사 분위기 반영)
- **✍️ Writer**: 회사 웹사이트의 주요 페이지(메인, 회사소개, 서비스/포트폴리오)에 들어갈 핵심 카피라이팅 및 구조 초안을 작성해 주세요. (핵심 키워드: .NET 컨설턴트, 자동화, 시스템 구축, 전문성)
