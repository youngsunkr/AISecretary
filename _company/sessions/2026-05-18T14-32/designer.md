## 🖥️ 웹사이트 메인 페이지 와이어프레임 설계 (Landing Page Blueprint)

**🎯 목표:** 방문자(잠재 고객/회사)에게 에프원소프트의 전문성, 기술 트렌드 선도력, 그리고 신뢰감을 즉각적으로 전달하고, 컨설팅 서비스로의 전환을 유도한다.

**✨ 핵심 디자인 원칙 적용:**
*   **톤앤매너:** 전문적이고 신뢰감 있으며 (Professional & Trustworthy), 미래 지향적인 기술 트렌드를 반영 (Futuristic/Modern).
*   **컬러 사용:** 브랜드 메인 컬러를 강조하여 시각적 계층 구조(Visual Hierarchy)를 명확히 한다.
*   **레이아웃 콘셉트:** Clean Architecture 및 Dashboard Kit의 구조를 차용하여 정보의 위계가 명확한 수직 흐름을 따른다.

### 1. Header (Sticky Navigation Bar)

| 요소 | 내용/설명 | 디자인 지침 |
| :--- | :--- | :--- |
| **Logo** | 에프원소프트 로고 | 좌측 상단 배치. 브랜드 메인 컬러 또는 흰색 배경 위에서 강조. |
| **Navigation (Nav)** | 홈, 서비스 소개, 기술 포트폴리오, 회사 소개, 문의하기 | 중앙 정렬 또는 우측에 배치. 메뉴는 간결하게 유지. |
| **CTA Button** | '무료 상담 신청' | 우측 끝에 배치. 브랜드 액션 컬러(Primary Color)를 사용하여 가장 눈에 띄게 처리. |

### 2. Hero Section (최상단 핵심 영역)

**목표:** 방문자가 스크롤을 내리는 즉시 회사의 정체성과 핵심 가치를 인지하게 한다.

| 요소 | 내용/설명 | 디자인 지침 |
| :--- | :--- | :--- |
| **Main Headline (H1)** | **"AI와 .NET 기반, 미래를 설계하는 기술 컨설팅"** | 가장 크고 굵게 처리. 회사의 핵심 가치(AI + .NET)를 명확히 제시. |
| **Sub-Headline (P)** | "복잡한 비즈니스 문제를 최신 기술 트렌드에 맞춰 해결하고, 지속 가능한 성장을 위한 로드맵을 제공합니다." | H1 아래에 배치. 신뢰감을 주는 문구로 전문성을 강조. |
| **Visual Element** | 추상적인 AI/데이터 흐름 또는 깔끔한 기술 아키텍처를 상징하는 모션 그래픽/고급 일러스트레이션. | 배경 또는 좌측에 배치하여 시각적 몰입도 극대화. (미래지향적 톤 강조) |
| **Secondary CTA** | '기술 컨설팅 서비스 보기' | 메인 CTA와 함께 배치. |

### 3. Trust & Credibility Section (신뢰 구축 영역)

**목표:** 잠재 고객에게 회사의 전문성과 신뢰도를 빠르게 입증한다.

| 요소 | 내용/설명 | 디자인 지침 |
| :--- | :--- | :--- |
| **Trust Bar** | 주요 클라이언트 로고 또는 수상 내역 (가상) | 3~4개의 핵심 아이콘과 함께 배치. 깔끔한 그리드 레이아웃 사용. |
| **Key Pillars** | 핵심 컨설팅 영역 3가지 (예: .NET 아키텍처 설계, AI 모델 통합, 시스템 최적화) | 각 기둥별로 아이콘과 짧은 설명으로 구성하여 시각적으로 분리. |

### 4. Service Showcase Section (서비스 상세 소개)

**목표:** 제공하는 컨설팅 서비스의 범위를 구체적으로 제시한다.

| 요소 | 내용/설명 | 디자인 지침 |
| :--- | :--- | :--- |
| **Section Title** | "우리가 제공하는 핵심 기술 솔루션" | 명확하게 구분. |
| **Service Cards (3~4개)** | 각 서비스(예: .NET 백엔드 설계, ML.NET 평가 시스템 구축, AI 전략 수립)를 카드 형태로 제시. | 각 카드에는 해당 서비스의 아이콘, 제목, 간결한 설명 포함. 마우스를 올리면(Hover) 인터랙션 추가 고려. |
| **Visual Flow** | 기술 스택( .NET, C#, ML.NET, Azure 등)을 시각적으로 연결하는 다이어그램 요소 삽입. | 복잡한 기술 내용을 직관적인 흐름으로 시각화하여 전문성을 강조. |
